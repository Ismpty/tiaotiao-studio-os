# Prompt Translation Rule（提示词转写规则）

内部记录可以写：

```text
DLHU827 reference
Source post 01
Candidate pattern
```

但模型可见 Prompt 必须写成自然语言。

不要只写：

```text
use DLHU827 style
follow creator reference
use Source Post 01
```

应转写为：

```text
Keep the same scene layout, lighting direction, main props and character appearance stable while changing the camera to a low side angle. The character remains in the same physical space, and the new view must feel like a real camera moved to another position, not a new scene.
```
