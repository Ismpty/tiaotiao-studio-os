# Definition（定义）

## 中文

Programmer Outfit 是 Jump 的默认身份服装，也是 TiaoTiao Studio 最具代表性的角色造型。

它代表的是一种现代创作者、程序员和数字工作者的生活方式，而不是传统意义上的职业制服。

## English

The Programmer Outfit is Jump's default appearance and the signature costume of TiaoTiao Studio.

It represents modern creators, software developers and digital professionals.

---

# Design Philosophy（设计理念）

服装应体现：

- 舒适
- 实用
- 简洁
- 年轻

它不是时装秀，而是每天都会穿的衣服。
