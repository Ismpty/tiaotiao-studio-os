# PROJ-001 — Video Production Pipeline（视频生产项目规范）

> **Canonical Project Record（官方项目规范）**

# Overview（概述）

## Purpose（用途）

### 中文

Project 是整个 Studio 的生产中心。

一个 Project 对应一个实际制作的视频或内容。

### English

A Project represents one complete production task, from idea to publishing.

---

# Production Pipeline（生产流程）

```text
Idea（创意）
        ↓
Story（故事）
        ↓
Storyboard（分镜）
        ↓
Prompt（提示词）
        ↓
Image / Video（素材生成）
        ↓
Editing（剪辑）
        ↓
Publishing（发布）
        ↓
Analytics（数据分析）
```

---

# Required Records（必须关联）

每个 Project 必须关联：

* Character
* Episode
* Story
* Environment
* Motion
* Camera
* Lighting
* Prompt
* Asset

---

# Deliverables（交付内容）

必须包含：

* 视频
* 封面
* 标题
* 描述
* 标签
* 发布记录

---

# Quality Checklist（质量检查）

发布前确认：

* 角色一致
* Prompt 正确
* Logo 正确
* 字幕正确
* 音乐版权正确
* 导出规格正确

---

# Publishing（发布）

支持平台：

* YouTube
* TikTok
* Instagram
* 小红书
* 抖音
* 视频号

---

# Success Metrics（成功指标）

记录：

* 播放量
* 完播率
* 点赞率
* 分享率
* 评论率
* 收藏率
