# 1. Purpose（目的）

## 中文

本文件用于指导《跳跳下班啦》试播项目的字幕内容、字幕时间点、字幕位置和字幕安全区。

字幕的作用不是解释所有画面，而是帮助观众快速理解：

- 跳跳完成了一天工作
- 电脑一关，生活开始
- 下班后的时间重新属于自己
- 《跳跳下班啦》的系列情绪

字幕必须温暖、简洁、生活化，不制造焦虑，不标题党，不遮挡 Jump。

## English

This file defines caption content, timing, placement and safety rules for PROJ-001 — Jump After Work Pilot Project.

Captions should support the story without over-explaining the image.

# 3. Caption Principle（字幕原则）

```text

[ ] 清晰
```

# 7. Caption Style（字幕样式）

## Recommended Style

```text
Font:
Clean modern Chinese font

Weight:
Medium / Semibold

Size:
Readable on mobile

Color:
Warm white / soft off-white

Optional Accent:
Fire Dragon Fruit Pink for small emphasis only
```

## Subtitle Background

推荐：

```text
Soft shadow
Subtle dark translucent background
No heavy box
No harsh outline
```

避免：

```text
粗黑描边
综艺花字
夸张描边
高饱和渐变字
儿童感字体
廉价营销风字幕
```

---

# 8. Caption Position Rules（字幕位置规则）

字幕必须放在：

```text
central 3:4 safe area
```

推荐区域：

```text
画面下方偏中
避开平台底部 UI
避开 Jump 的脸
避开 Jump 的前爪动作
避开关键叙事物件
```

禁止：

```text
贴近最底部
贴近最顶部
覆盖 Jump 的脸
覆盖 Jump 的手
覆盖电脑屏幕关键内容
覆盖背包
覆盖门口动作
```

---

# 9. Caption Duration Rules（字幕时长规则）

每条字幕建议：

```text
Minimum:
1.8 seconds

Recommended:
3–6 seconds

Maximum:
8 seconds
```

规则：

```text
[ ] 字幕出现时间要足够读完
[ ] 不要频繁闪现
[ ] 不要一镜头出现太多文字
[ ] 每个镜头最多 1 条主字幕
[ ] 结尾字幕可以稍微停留更久
```

# 11. Voiceover and Caption Relationship（旁白与字幕关系）

如果使用旁白，字幕不需要完全重复旁白。

## Voiceover

```text
今天的代码，终于写完了。

跳跳看了一眼窗外，突然想出去走走。

不一定要去很远的地方。

有时候，一点晚风就够了。

忙了一天，也该把时间还给自己一点点。

下班啦，生活开始啦。
```

## Caption

```text
今天的代码写完了
电脑一关，生活上线
想出去走走
不远，也可以是冒险
把时间还给自己一点点
下班啦，生活开始啦
```

原则：

```text
旁白负责情绪。
字幕负责记忆点。
画面负责故事。
```

# 14. Platform Caption Notes（平台字幕备注）

## 小红书

```text
字幕可以稍微温暖、有生活感。
建议保留完整字幕。
```

## 抖音

```text
字幕要更短、更清晰。
前 3 秒字幕必须让用户知道内容。
```

## 视频号

```text
字幕语气要自然，不要太网感。
适合温暖、轻共鸣。
```

## YouTube Shorts / TikTok / Reels

```text
可以使用英文字幕版本。
保持简单，避免长句。
```

# 18. COMMAND-005 Caption Check（字幕一致性检查）

最终导出前，字幕部分需要通过：

```text
COMMAND-005 — Run Consistency Check
```

检查对象：

```text
Caption Text
Caption Timing
Caption Placement
Caption Tone
```

允许结果：

```text
PASS
PASS WITH MINOR FIXES
```

如果结果为：

```text
FAIL
```

必须修改字幕后重新检查。
