# 1. Purpose（目的）

## 中文

本文件用于指导《跳跳下班啦》试播项目的最终剪辑装配。

当 Shot 1–8 的视频片段通过审片后，剪辑应按照本文件进行组装，确保：

- 镜头顺序正确
- 故事情绪完整
- Jump 角色连续
- 动作衔接自然
- 字幕不遮挡主体
- 音乐和音效服务情绪
- 最终成片符合 TSOS Source of Truth

## English

This file defines the editing assembly guide for PROJ-001 — Jump After Work Pilot Project.

After Shot 1–8 clips are approved, the final edit should be assembled according to this guide.

# 8. Shot Assembly Rules（镜头装配规则）

## Shot 1 → Shot 2

```text
连接逻辑：
从“工作结束”进入“决定下班”。

剪辑要求：
Shot 1 结尾保留 Jump 手离开键盘的动作。
Shot 2 开头接 Jump 看向窗外或离开办公室动作。

避免：
不要直接从键盘动作跳到门口。
不要让电脑状态突然变化。
```

---

## Shot 2 → Shot 3

```text
连接逻辑：
从“决定离开”进入“收拾东西”。

剪辑要求：
Shot 2 中电脑关闭后，Shot 3 展示背包和桌面物件。
保持桌面和灯光连续。

避免：
不要让电脑又突然变成打开状态。
不要让背包突然消失。
```

---

## Shot 3 → Shot 4

```text
连接逻辑：
从“收拾完成”进入“离开工位”。

剪辑要求：
Shot 3 结束时背包状态明确。
Shot 4 开始时 Jump 已携带背包。

避免：
不要让背包样式明显变化。
不要让 Jump 位置瞬移感太强。
```

---

## Shot 4 → Shot 5

```text
连接逻辑：
从“走向出口”进入“开门离开”。

剪辑要求：
Shot 4 结尾应朝出口方向移动。
Shot 5 开头应在门口或接近门口。

重点：
这是最重要的空间连续性连接。

避免：
不要让运动方向反转。
不要让办公室空间完全换掉。
```

---

## Shot 5 → Shot 6

```text
连接逻辑：
从“打开门”进入“下班后的暖光”。

剪辑要求：
Shot 5 结尾有门外暖光。
Shot 6 开头延续暖光和开放感。

避免：
不要突然变成完全不同地点。
不要过曝。
不要变成广告摆拍。
```

# 9. Transition Rules（转场规则）

推荐使用：

```text
Hard cut
Soft cut
Cut on action
Natural motion transition
Light transition
```

避免使用：

```text
炫技转场
模板化转场
闪白转场
旋转转场
快闪卡点
过度缩放
综艺感转场
```

本片应该像温暖的生活纪录片，而不是短视频模板剪辑。

# 10. Caption Placement Rules（字幕位置规则）

字幕必须：

```text
[ ] 放在安全区内
[ ] 不遮挡 Jump 的脸
[ ] 不遮挡 Jump 的前爪和关键动作
[ ] 不遮挡背包、电脑、食物、购物篮、屏幕光等关键叙事物件
[ ] 不超过画面承载
[ ] 字体清晰
[ ] 语气温暖
[ ] 不标题党
[ ] 不制造焦虑
```

推荐位置：

```text
画面下方偏中
避开平台 UI
避开 Jump 主体
```

# 16. Opening 3 Seconds Check（前三秒检查）

开头 3 秒必须让用户理解：

```text
这是一个工作结束的场景。
Jump 是主角。
故事即将从工作切换到生活。
```

检查项：

```text
[ ] Jump 是否清楚出现
[ ] 办公桌 / 电脑是否清楚
[ ] 工作结束状态是否明显
[ ] 画面是否温暖
[ ] 没有信息混乱
```

---

# 17. Ending Check（结尾检查）

结尾必须完成情绪落点：

```text
下班不是结束，是生活重新开始。
```

检查项：

```text
[ ] Shot 6 是否有温暖停留
[ ] 结尾字幕是否清楚
[ ] 音乐是否有轻微收束
[ ] 画面是否适合系列化
[ ] 没有硬广感
```

---

# 18. Rough Cut Checklist（粗剪检查）

粗剪完成后检查：

```text
[ ] Shot 1–8 顺序正确
[ ] 总时长约 45 秒
[ ] 每个镜头都有叙事功能
[ ] Jump 角色没有明显跳变
[ ] 背包状态连续
[ ] 电脑状态连续
[ ] 光线变化自然
[ ] 镜头没有明显断裂
[ ] 字幕不遮挡主体
[ ] 音乐方向正确
[ ] 整体情绪温暖轻松
```

---

# 19. Fine Cut Checklist（精剪检查）

精剪完成后检查：

```text
[ ] 开头 3 秒清楚
[ ] 中段节奏不拖沓

[ ] Shot 6 情绪停留足够
[ ] 字幕节奏和画面匹配

[ ] 结尾有记忆点
[ ] 不像广告

[ ] 不制造焦虑
```
