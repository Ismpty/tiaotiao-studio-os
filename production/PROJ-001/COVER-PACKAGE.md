# 1. Production Purpose（生产目的）

## 中文

封面目标是：

- 一眼看出“跳跳下班啦”
- 保持 Jump 角色一致性

- 能作为《跳跳下班啦》系列封面模板延展

## English

This file defines the production-ready cover package for PROJ-001 — Jump After Work Pilot Project.

The cover should clearly communicate the Jump After Work series identity while preserving character, brand and visual consistency.
