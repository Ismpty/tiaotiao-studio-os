# 1. Purpose（目的）

## 中文

本文件用于在 Shot 1–6 都完成生成和审片后，记录最终进入剪辑的片段版本。

它的作用是：

- 确认每个镜头最终选用哪个版本
- 记录为什么选择该版本
- 检查 6 个镜头之间是否连续
- 检查最终片段是否可以进入剪辑
- 避免把未通过审片的片段误放入最终剪辑

## English

This file records the final approved clip selection for PROJ-001.

It confirms which generated clip version should enter the final edit and why.

# 3. Final Selection Rule（最终选择规则）

如果同一镜头有多个通过版本，优先选择：

```text
1. Jump 角色最稳定的版本
2. 动作最自然的版本
3. 与上下镜头最容易衔接的版本
4. 灯光最符合整体风格的版本
5. 情绪最温暖的版本
6. 画面最清晰的版本
```

不得选择：

```text
角色身份错误的片段
Jump 变成人类或其他动物的片段
脚步严重滑动的片段
身体漂浮的片段
风格偏赛博朋克 / 恐怖 / 游戏感的片段
无法与上下镜头衔接的片段
```
